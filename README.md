# 発電機起動停止計画（ユニット・コミットメント）例題

調査書に基づく **Unit Commitment (UC)** の数理モデルを、Python で MILP（混合整数線形計画）として実装した例題です。

## 必要な環境

- Python 3.8+
- PuLP（およびデフォルトで同梱される CBC ソルバー）

```bash
pip install pulp numpy matplotlib
```

## 実行方法

```bash
python main.py
```

計画結果の図は **`results/`** フォルダに保存されます。ファイル名は **解析年月日(YYMMDDHHMMSS)＋解析条件**（例: `250309143052_3d_s2_m1_solar1_wind1.png`）です。

## モデルの概要（調査書との対応）

| 項目 | 内容 |
|------|------|
| **目的関数** | 燃料費（線形）＋無負荷コスト＋起動コストの最小化 |
| **変数** | 3変量モデル: \(u_{g,t}\)（稼働）, \(v_{g,t}\)（起動）, \(w_{g,t}\)（停止）, \(P_{g,t}\)（出力） |
| **需給バランス** | \(\sum_g P_{g,t} = L_t\) |
| **予備力** | \(\sum_g (P_{g,max} \cdot u_{g,t} - P_{g,t}) \geq SR_t\) |
| **出力上下限** | \(P_{g,min} \cdot u_{g,t} \leq P_{g,t} \leq P_{g,max} \cdot u_{g,t}\) |
| **状態遷移** | \(u_{g,t} - u_{g,t-1} = v_{g,t} - w_{g,t}\) |
| **最低運転時間 (MUT)** | 起動後 MUT 時間は稼働継続 |
| **最低停止時間 (MDT)** | 停止後 MDT 時間は停止継続 |
| **ランプ制約** | 隣接時刻間の出力増減を RU/RD で制限 |

## 対話形式で指定するパラメータ

実行時に以下を入力します（Enter で省略時は括弧内のデフォルト）。

| 項目 | 単位/選択肢 | 説明 |
|------|-------------|------|
| **解析対象日数** | 1〜7 日 | 計画期間（1日=24時間、最大7日） |
| **解析対象季節** | 1=春秋, 2=夏, 3=冬 | 需要曲線のパターン |
| **太陽光発電の曲線** | 1=快晴日, 2=準快晴日, 3=曇天日, 4=雨天日, 5=ランプアップ, 6=ランプダウン | 1日24時間の太陽光出力パターン |
| **風力発電の曲線** | 1=風が強い日, 2=風が弱い日, 3=午前中が強い日, 4=午後が強い日 | 1日24時間の風力出力パターン |
| **日ごとの再エネ条件** | 1=全日同じ, 2=日ごとランダム, 3=シナリオ（日別指定） | **解析対象が2日以上のときのみ**、日数入力の直後に選択 |

- **1=全日同じ**: このあと入力する「太陽光曲線」「風力曲線」を全期間でそのまま繰り返し。
- **2=日ごとランダム**: 各日ごとに太陽光・風力のタイプをランダムに決定。**太陽光・風力の単一指定は不要**（入力は求めません）。
- **3=シナリオ**: **日別に指定する対話**が入る。各日の太陽光曲線(1–6)・風力曲線(1–4)をカンマ区切りで入力（例: 3日なら 「1,2,3」と「1,1,4」）。
- 解析対象が1日のときはモード選択は行わず、太陽光・風力曲線を入力してその日のみに適用する。

需給バランスは **純負荷（需要 − 太陽光 − 風力）** を火力で賄う形で計算します。

## データ

- **発電機**: 10機（G1〜G10）、各機の Pmin/Pmax、コスト、MUT/MDT、ランプ率を定義
- **需要**: 季節別の1日パターンを日数分繰り返し
- **太陽光・風力**: 選択した曲線を日数分繰り返し
- **予備力**: 純負荷の 10% を必要予備力として設定

## 図の出力（PNG）

求解後に `unit_commitment_result.png` を出力します。

1. **需給バランス**: 需要、太陽光・風力、純負荷、火力合計の時系列（複数日時は日境界に破線）
2. **機別火力出力（積み上げ）**: 各発電機の出力を積み上げたスタック図と純負荷曲線
3. **稼働状態ヒートマップ**: 各機のオン/オフ（緑=稼働、赤=停止）を時刻別に表示

## 出力例

- ステータス・総コスト
- 時刻別の稼働状態 \(u\) と発電出力 \(P\)
- 需給・予備力の確認
- 起動/停止が発生した時刻
- コスト内訳（燃料・無負荷・起動）
